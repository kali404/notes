## setoolkit
社会工程师工具包(SET)是由 TrustedSec 的创始人 Dave Kennedy 创建和编写的。 它是一个开源的 python 驱动的工具，旨在围绕社会工程进行渗透测试。
社会工程师工具包(SET)是一个为社会工程设计的开源渗透测试框架。 Set 有许多自定义攻击向量，允许你在一小部分时间内进行一次可信的攻击。 这些工具利用人类的行为来欺骗他们找到攻击向量。
