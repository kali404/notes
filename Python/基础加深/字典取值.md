## Python 字典

### setdefault() 方法

setdefault() 方法和 get()方法类似,返回指定键的值，如果键不在字典中，将会添加键并将值设置为一个指定值，默认为None。

### 字典的遍历

 ```python
info = {
    'name':'xiaoming',
    'sex':'nan',
    'age':20,
    'id':1
}

info2 = {
    'name':'hhh',
    'sex':'nv',
    'addr':'beijign'
}
 ```

一、

```python
# 直接循环字典，循环的是字典的key，如果想循环的时候同时获取到key和value，那么就要用 .items()方法
for k,v in info.items():
    print('%s is %s'%(k,v))
```

二、

```python
for k in info2: # 这种方式效率比较高
    print(k,info2[k])
```
